How to use this tool
Read before posting - tap to expand

Each member has 3 contact cards with their details laid out in a different order, plus 12 example messages for inspiration. The best referrals are always written in your own words - use the contact cards as your starting point and build your message from scratch around them.

The recommended approach
  1. Copy a contact card (variations 1, 2 or 3) - each one has the details in a different order
  2. Write your own message in the comment or post field, in your own voice
  3. Weave the contact details in naturally as you write - phone number early, website or email later, or however it flows
  4. Read it back and ask yourself: does this sound like me? If yes, post it
  • ✏️
    Write it yourself wherever possible
    Mention how you know them, something specific they did well, or a client you referred to them. One genuine sentence in your own words is worth more than a polished message that does not sound like you.
  • 📋
    Use the example messages as inspiration only
    Variations 4 to 15 are there if you are stuck. Rewrite them in your own voice before posting - change the structure, swap out phrases, make it yours. Never copy one word for word.
  • 🚫
    Never post the same message twice
    If multiple members are referring the same person, your messages should look nothing alike. If you see someone has already posted about a member, write yours from scratch rather than using a similar template.

The goal is a referral that sounds like you. Someone reading your post should think "that sounds exactly like something they would say" - not "that looks like it came from a template." The contact card gives you the details. The rest should be you.
